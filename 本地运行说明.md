# 基于区块链的食用油运输监管系统

## 环境要求

- Windows 10/11
- Docker Desktop（需要启动）
- Go 1.20 或更高版本
- Node.js 16 或更高版本

## 一键启动

首次安装 Docker Desktop 后，如果提示 WSL2 未启用，请以管理员身份运行：

```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\setup-docker.ps1
```

完成后重启电脑并启动 Docker Desktop。

在项目根目录打开 PowerShell：

```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\start-local.ps1
```

脚本会自动启动 MySQL、创建数据库及初始数据、构建前端并启动后端。浏览器访问：

```text
http://127.0.0.1:9090
```

## 停止与重置

```powershell
.\stop-local.ps1
.\reset-demo-data.ps1
```

重置会删除本地演示数据库。再次启动时会重新生成完整演示数据。

## 区块链存证说明

系统业务层保留了完整的区块链存证语义，包括数据哈希、前置存证哈希、交易哈希和区块哈希。
当前本地演示运行模式使用存证服务生成并持久化哈希链，不要求启动 Fabric 网络；后续可将存证服务替换为 Fabric Gateway 实现，而无需重做业务页面和流程。
